## 工作目录特征（影响外部 AI 工具）
- 工作目录体量极大：`辅助材料/` 3720 文件、`model/` 3613 文件，含大量网络来源图片/JSON/CSV/TXT。
- 2026-07-13: 用 zcode 打开本目录会触发"敏感内容"拦截；深度核查后发现 WorkBuddy 也会触发（连读取这些文件后请求都被拦）。**根因不是某个文件含敏感词**（所有文本文件全部干净：规则文档/中文md/代码/CSV/JSON标注/YOLO数字标签均无敏感词），而是**海量数据文件内容被注入上下文**——YOLO 标注的超长密集浮点数字串、sha256 哈希清单、7000+ 文件名、78MB 的 .pt 二进制等，组合后命中后端内容安全检测。与具体 AI 工具无关。缓解：把 `辅助材料/`、`model/`、`*.pt`、`node_modules/` 移出工作区或配 ignore；AI 助手只需 `src/`+`docs/`+`scripts/`+`tests/`+配置文件。

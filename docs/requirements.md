# 开发需求文档 — 甲如

## 1. 项目概述

**项目名称：** 甲如 (JiaRu)
**产品定位：** Web端美甲试色应用，无需下载安装，手机/电脑浏览器直接访问
**目标用户：** 美甲店顾客、美甲爱好者
**核心价值：** 解决用户在美甲店选款困难的问题，直观预览美甲效果

## 2. 功能需求

### Phase 1: MVP 极简版（当前阶段）

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 首页引导 | P0 | 品牌展示 + 四个功能入口（上传试色、图库、AI生成、AR试戴） |
| 照片上传 | P0 | 支持本地上传和拍照（移动端） |
| 涂色编辑器 | P0 | 在照片上选择颜色涂抹在指甲位置 |
| 颜色选择 | P0 | 20种预设流行色 + 自定义取色器 |
| 保存图片 | P0 | 将涂色结果保存为图片到本地 |
| 撤销/重置 | P0 | 撤销最近操作、重置全部效果 |
| 预设图库 | P1 | 浏览预设美甲款式图 |
| 移动端适配 | P0 | 触屏友好，按钮大，操作简单 |

### Phase 2: AI 生成模式

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 文字描述输入 | P0 | 用户输入风格描述词 |
| 风格预设选择 | P0 | 10种预设风格一键选择 |
| AI图片生成 | P0 | 调用DALL-E 3生成美甲效果图 |
| 结果展示与保存 | P0 | 展示AI生成结果并支持保存 |

### Phase 3: AR 实时试戴

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 摄像头调用 | P0 | 请求并获取摄像头权限 |
| 手部追踪 | P0 | MediaPipe实时检测手部 |
| 指甲贴合 | P0 | 在检测到的指甲位置叠加颜色 |
| 实时跟随 | P0 | 美甲效果随手指移动实时更新 |

## 3. 用户故事

### MVP 用户旅程
```
用户打开网站 → 看到"甲如"品牌首页 → 点击"上传照片试色"
→ 选择/拍摄手部照片 → 照片显示在编辑器中
→ 从底部颜色面板选择喜欢的颜色 → 在指甲上点击涂抹
→ 使用撤销/重置调整效果 → 满意后点击保存
→ 保存图片到本地 → 分享给美甲师
```

## 4. 验收标准

### MVP 验收
- [ ] 在手机浏览器中打开首页，所有按钮可点击
- [ ] 上传照片后编辑器正常显示
- [ ] 选择颜色后在指甲区域点击有涂抹效果
- [ ] 撤销按钮能回退上一步操作
- [ ] 保存按钮能下载涂色后的图片
- [ ] 页面在手机竖屏模式下布局正常
- [ ] 所有操作在5秒内给出反馈
